第零章第一节 教程简介¶

原文作者：Alex

首次发布：2007 年 5 月 27 日

最近更新：2020 年 5 月 18 日

译者：Karl Liu

翻译时间：2021 年 1 月 31 日

原文链接：https://www.learncpp.com/cpp-tutorial/introduction-to-these-tutorials/

热烈欢迎！¶

欢迎来到这篇教程！这篇教程最重要的目标是让学习 C++ 变得简单。

不像很多其他的网站或者书记，本教程不要求你拥有任何编程经验。我们会在你逐渐进步的过程中通过 大量 的例子教会你所有东西。

无论你是以 C++ 作为爱好还是希望成为专业的 C++ 程序员，本教程都很适合你！

课程结构¶

这个介绍章节中的课程的主要目标是让你了解一些诸如什么是 C++、它是如何出现的、程序是如何工作的以及你需要为了创造自己的程序需要安装什么软件这样的背景知识。同时你也会写出你自己的第一个程序。

之后的章节将会带你探索 C++ 的不同部分。在第一章中，你会对 C++ 的很多基础概念产生一个浅显的认识，这样我们就可以开始写一些简单的程序。之后的章节会对这些基础概念逐一深入，或引入一些新的概念。

每个章节都有一个主题，其中的每一节都会大体上和这个主题相关。本教程中的各个章节均没有推荐阅读时间，按照一个你舒适的速度进行学习即可。

目标¶

在我们开始之前，我们要先聚焦一下本教程的一些目标：

  • 不仅仅有 C++，还涵盖其他编程相关的话题。传统教科书在教授给定编程语言的基础知识方面做得很好，但它们通常不涵盖该语言附带的相关编程主题。例如，书籍将省略有关编程风格，常见陷阱，调试，好或不好的编程实践和测试的部分。因此，当你读完这本教材之后，你可能已经了解了如何使用这门语言进行编程，但是同时还养成了不良习惯，这些习惯在未来会给你带来很多困扰！本教程的目标之一是确保合适的地方自然地涵盖所有这些附带的主题。完成本教程后，你不仅会知道如何使用 C++ 进行编程，而且还将知道如何不使用 C++ 进行编程，这无疑是同样重要的。

  • 提供大量示例。比起文本描述，大多数人可以在实例中学到更多的东西。因此本教程将尽可能提供大量清晰、简洁的示例，以展示如何应用你正在学习的概念。我们还将尽可能避免孪生恶魔：神奇的金手指，指为了节省空间，省略了示例的一部分；或者在示例中引入新概念而不详细说明。这两者都很容易导致读者无法理解文章内容。

  • 提供练习程序。在很多课程和章节的末尾本教程会提供一些练习题。你可以选择尝试着解决这些练习题。他们带有答案，你可以对比自己的作答与正确答案之间的区别，也可以在被卡住的时候看一看我们的解法。之后你可以返回到你觉得你需要复习的部分。

  • 最重要的是：有趣！编程是一件很有趣的事情。如果你在编程的过程中没有感受到趣味，那你的心境一定出了问题。疲劳和不开心的程序员经常会犯错，而纠错通常要比在一开始写的时候就写对要浪费更多的时间。所以“遇到困难睡大觉”通常可以为你节省不少时间。

充分利用本教程¶

在你阅读本教程时，我们推荐你做以下的事情来最大化你的学习体验：

  • 动手抄写示例程序并且自己编译它们。不要复制粘贴！这可以帮助你了解一些大家都会犯的错误，并且可以熟悉编译告警和错误。不要无脑地抄这些代码，最好想一想写每一行在做什么，以及这每一行对程序的功能做出的贡献。如果你发现了什么讲不通的或者无法理解的东西，这就是你可以深入研究的部分。

  • 在你犯错或者在自己写的程序里找到漏洞的时候把它们及时修正掉。尽量独立地解决自己的问题。学习如何寻找并修正错误对于编程来说是一个很关键的技能。不要忘了学习如何使用调试器（在后面的章节中我们会解释），这是个可以帮助找出程序哪里出了错的关键的工具。

  • 在示例的基础上做一些实验。改一改程序中的数字或者文本看看会发生什么。修改程序使得它可以做一些额外的事情（比如如果一个示例程序的功能是求出两数之和，你就可以试着让他能够求出三数之和）。相比于简单地跑一遍示例程序，修改一下它们可以让你学到更多东西。

  • 做好心理预期，课后习题可能会花一点时间。如果你是个编程新手，你可能觉得这些习题很有挑战性（这是很正常的，你的大脑需要适应一下编程思维模式）。如果你一开始没有得到正确答案，不要气馁。你可能需要尝试几种不同的方式才能最终成功。如果你实在卡住了看一下题解也是可以的。但一定要确保你在继续前搞懂答案是如何解决这个问题的。

  • 运用你学到的知识写一些自己的小程序。实践出真知。

常见的网站相关的问题¶

问：怎么注册登录该网站？

没必要，任何人都可以匿名地访问本网站的任何部分。

问：这些教程最初是在 2007 年写的，现在他们还有用吗？

当然是的。

C++ 一般不会随着时间出现什么变化，但当它确实变了的时候，我们会添加大量新内容。本教程一直在根据语言的变化和读者的反馈时常更新。

问：这个网站有没有可以在线下看的 PDF 版本？

很遗憾，没有。这个网站之所以可以保持免费是因为我们被广告资助了。这个商业模式在 PDF 格式下很难搞。你可以处于私人目的自己将这个网站转换成 PDF 格式，只要你不发布这些内容。

问：如果我卡在了一个概念过不去该怎么办？

如果你搞不懂什么问题：

  • 看看评论区，其他读者可能遇到了类似的问题。

  • 看看下一个章节，你的问题可能会在后面得到回答。

  • 百度/谷歌/DDG 一下你就知道。

  • 在编程问答网站上提问，比如 Stack Overflow。

如果这些办法都没能奏效，跳过你不明白的内容，以后再回来看。你可能会发现在习得了额外的知识并了解了其他文章中的内容后，这个部分就不再那么难以理解了。

问：如果我忘记了某个东西的含义怎么办？

使用网站导航。在里面你可以查找到任何你想要知道的话题，同时你会找到指向讨论这个话题的文章的链接。

问：我怎么才能在评论区里使用头像？

评论区使用了 gravatars。你可以在 gravatar.com 中创建一个。点击“创建自己的 gravatar”按钮并注册，你的 gravatar 就会被连接到你的邮箱地址。

问：你可以做一个深夜模式吗？

不太容易，但是你可以。参见 darkreader.org。

最后，唠叨一句：这个网站之所以免费是因为它通过广告盈利。如果你喜欢这些内容，请考虑关掉广告拦截器。（译注：所以看译文时请先点一下原链接）

好了，我们开始吧！
