教程简介¶

热烈欢迎！¶

欢迎来到这篇教程！这篇教程最重要的目标是让学习 C++ 变得简单。

不像很多其他的网站或者书记，这篇教程不要求你拥有任何编程经验。我们会在你逐渐进步的过程中教会你所有东西，同时还包括了大量的例子。

无论你是以 C++ 作为爱好还是希望成为专业的 C++ 程序员，这篇教程都很适合你！
